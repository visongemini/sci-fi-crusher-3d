# 星际粉碎机 3D

科幻风格的 3D 解压小游戏。和《末日之战》一样，项目使用 HTML + JavaScript；3D 画面使用 Three.js。

## 运行方式

直接双击 `index.html`，用 Edge 或 Chrome 打开即可。

也可以双击 `启动星际粉碎机3D.bat`，它会启动本地服务并自动打开游戏。玩完后双击 `停止星际粉碎机3D.bat` 可以关闭这个本地服务。

公网部署时可以直接使用仓库根目录作为静态网站目录。`index.html` 是入口文件，`vendor/three.min.js` 已经放在项目里，不依赖外部 CDN。

如果浏览器限制本地文件，可以在这个目录运行：

```powershell
python -m http.server 8788
```

然后打开：

```text
http://localhost:8788/
```

## 操作

- 鼠标或触摸点击传送带上的物品。
- `1` 切换粉碎机：把物品吸进机器压碎。
- `2` 切换锤子：直接把物品砸爆。
- `3` 切换激光：用能量束切碎物品。
- `4` 切换重压器：从上面压扁后爆碎。
- `5` 切换电锯：高速切割后爆开。
- `6` 切换冰爆：先冻结再炸裂。
- `7` 切换磁撕：用磁力撕开物品。
- `8` 切换黑洞：压缩到极限再爆碎。
- `P` 暂停或继续。
- `R` 重新开始。

## 新玩法

顶部有升级商店。玩家可以用分数购买更精细的高级物品：

- 精密机器人
- 微型机甲
- 反应炉核心
- 星舰模型

买下高级物品后，它会进入展示栏收藏，同时也会加入传送带。高级物品被粉碎后给更多分数，所以游戏会越玩越快。

高级物品现在使用更公平的随机规则：不会因为刚买过某个物品，就让它连续霸占传送带；系统会尽量避开最近出现过的物品。

## 说明

游戏会从传送带送出能量核心、数据方块、无人机舱、钛合金桶、等离子电池、记忆棱晶等普通物品。粉碎或锤碎时会生成爆碎碎片和火花，连击越高加分越多。
